# Telegram Bot с переключением режимов

Бот с двумя режимами работы: **режим чата** и **режим терминала**.

## Возможности

### 💬 Режим чата
- Обычное общение с ботом
- Бот отвечает текстовыми сообщениями
- Команды сервера не выполняются

### 🖥 Режим терминала
- Любой текст воспринимается как Bash-команда
- Команды выполняются через `subprocess`
- Возвращается результат (stdout/stderr) в моноширинном формате
- Таймаут выполнения: 30 секунд

## Безопасность

✅ Только администратор (указанный в `ADMIN_ID`) может:
- Переключать режимы
- Выполнять команды в режиме терминала

❌ Все остальные пользователи получат сообщение "У вас нет доступа к этому боту"

## Использование

### Запуск бота

```bash
python bot_with_modes.py
```

### Переключение режимов

1. Нажми кнопку **"🖥 Режим терминала"** для активации режима терминала
2. Нажми кнопку **"💬 Режим чата"** для возврата в режим диалога

### Примеры команд в режиме терминала

```bash
ls -la
pwd
df -h
free -h
systemctl status xray
journalctl -u xray --since "10 min ago"
ps aux | grep python
```

## Настройка

Убедись, что в `.env` файле указаны:

```env
BOT_TOKEN=your_bot_token_from_botfather
ADMIN_ID=your_telegram_id
```

## Визуализация режимов

При переключении бот присылает подтверждение:

**Режим терминала:**
```
🖥 Режим терминала активирован

Теперь все твои сообщения будут выполняться как Bash-команды на сервере.
```

**Режим чата:**
```
💬 Режим чата активирован

Теперь я просто отвечаю на твои сообщения.
```

## Формат вывода команд

```
✅ Команда выполнена

Код возврата: 0

STDOUT:
```
(результат команды)
```

STDERR:
```
(ошибки, если есть)
```
```

## Ограничения

- Таймаут выполнения команды: 30 секунд
- Максимальная длина вывода: ~4000 символов (обрезается автоматически)
- Команды выполняются с правами пользователя, от которого запущен бот

## Интеграция с основнымботом

Если хочешь добавить эту функциональность в `main.py`, можно:

1. Скопировать логику переключения режимов
2. Добавить обработчик текстовых сообщений с проверкой режима
3. Интегрировать клавиатуру с существующими командами
